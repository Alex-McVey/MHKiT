API documentation
--------------------

The package contains the following modules:

* :ref:`qc_api_matlab`: Perform quality control analysis
* :ref:`utils_api_matlab`: Includes helper functions
* :ref:`wave_api_matlab`: Calculate quantities of interest for wave energy converters (WEC)
* :ref:`river_api_matlab`: Calculate quantities of interest for river energy converters (REC)
* :ref:`tidal_api_matlab`: Calculate quantities of interest for tidal energy converters (TEC)
